# Foothill High School Supermileage Club  
## Electronics System – Shell Eco-Marathon Vehicle (Prototype)

## ⚡ Overview
This repository contains all electronics-related design, development, and testing for the Foothill High School Supermileage Club’s Shell Eco-Marathon vehicle.

It focuses on the vehicle’s electrical and control systems, including the ECU, wiring, sensors, and data acquisition.

---

## 🎯 Objectives
- Develop a reliable and efficient electronic control system  
- Enable precise fuel and engine management  
- Collect and analyze performance data  
- Ensure safe and robust electrical design
